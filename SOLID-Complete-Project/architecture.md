                     +----------------------------+

                     | - String id                |                 |
                     | - String name              |                 |
                     | - List<Course> enrolledCs  |                 |
                     +----------------------------+                 |
                     | + enroll(Course)           |                 |
                     +-------------+--------------+                 |
                                   ^                                | (extends)
                                   | (extends)                      |
                 +-----------------+-----------------+              |
                 |                                   |              |
    +--------------------------+        +--------------------------+|
    |      CreditStudent       |        |       AuditStudent       ||
    +--------------------------+        +--------------------------+|
    | - Map<Course,Str> grades |        | (بيحضر كورس بس ملوش درجات)||

    | + addGrade(Course, Grade)|                                    |
    | + getGrades()            |                                    |
    +--------------------------+                                    |
                 |                                                  |
                 v (implements)                                     |
    +--------------------------+                                    |
    |       <<Gradable>>       |                                    |
    |     (Interface/ISP)      |                                    |
    +--------------------------+                                    |

## ⚙️ 2. مخطط الاعتماديات والخدمات (Dependency Setup Flow)

بيوضح إزاي كلاسات منطق العمل معزولة عن المخازن المادية باستخدام الـ Interfaces والـ DI:

```text
            +-----------------------------------------------+
            |                   main.dart                   |
            |     (بيعمل Instances ويحقنها في الكونسركتورز)    |
            +-----------------------+-----------------------+
                                    |
            +-----------------------+-----------------------+
            | (Injects Repo)                                | (Calls assignGrade)
            v                                               v
+-----------------------+                       +-----------------------+
|    StudentRegistry    |                       |       GradeBook       |
+-----------+-----------+                       +-----------+-----------+
            |                                               |
            | (Depends on)                                  | (Checks)
            v                                               v
+-----------------------+                       +-----------------------+
| <<StudentRepository>> |                       |     <<Gradable>>      |
|      (Interface)      |                       |   (Interface/ISP)     |
+-----------------------+                       +-----------+-----------+
            ^                                               ^
            | (Implements)                                  | (Implements)
+-----------------------+                       +-----------+-----------+
|InMemoryStudentRepos.  |                       |     CreditStudent     |
|     (Infrastructure)  |                       +-----------------------+
+-----------------------+
```

## 🔄 3. تدفق البيانات أثناء رصد درجة (Data Flow: Grading)

```text
[User input in main] ──(Student, Course, Grade)──> [GradeBook Service]
                                                           │
                                            (Is Student type Gradable?)
                                                           │
                                    ┌──────────────────────┴──────────────────────┐
                                    │ (Yes)                                       │ (No)
                                    v                                             v
                           [Cast to Gradable]                            [Print Safety Message]
                        (student.addGrade())                             (Audit Student Exit)
```

بكدة البنية المعمارية مبسطة جداً، واضحة، وخالية من أي حشو غير ضروري لتسهيل الفهم والمراجعة.
